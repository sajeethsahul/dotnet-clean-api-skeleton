# .NET Clean API Skeleton

Reusable **Clean Architecture API skeleton** built with **.NET 8, EF Core, and Swagger**.

This repository serves as a **starting point** for building clean, scalable, and maintainable backend APIs.  
It is intentionally **minimal** and **opinionated**, so teams can add features without fighting legacy complexity.
